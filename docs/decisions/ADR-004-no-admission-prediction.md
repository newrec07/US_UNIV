# ADR-004 — 합격·불합격 예측 기능 없음

- 상태: Accepted
- 결정일: 2026-06-22

## 맥락

대학 입학 결과는 정확한 개인별 확률로 계산할 수 없고, 거짓 정밀도는 학생의 대학 목록과
재정 의사결정에 피해를 줄 수 있다.

## 결정

- 합격·불합격 확률 또는 개인별 예측값을 만들지 않는다.
- 대학의 공개 통계는 연도와 출처를 포함한 객관적 참고 자료로만 제공한다.
- 학생 데이터는 학업, 시험, 활동, 일정, 재정 현황과 다음 행동을 분석하는 데 사용한다.

## 결과

- `StudentCollege.admitChance`를 도메인에서 제거한다.
- reach/target/safety 표현은 합격 판정으로 오해될 수 있어 별도 정책 전에는 사용하지 않는다.
