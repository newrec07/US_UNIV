# ADR-003 — 민감 원본 문서 저장

- 상태: Deferred
- 결정일: 2026-06-22

## 맥락

세금 신고서와 성적표에는 미성년자 신원정보와 가족 재정정보가 들어 있다.
원본 저장 여부에 따라 암호화, 접근 로그, 보존 기간, 삭제, 백업 정책이 크게 달라진다.

## 현재 결정

- 구조화된 학생·재정 데이터는 서버 DB에 저장한다.
- Messenger 대화와 사용자 의사결정도 서버에 저장한다.
- 원본 파일 업로드와 장기 보존은 해당 기능 구현 전에 다시 결정한다.
- 저장한다면 일반 DB 필드와 분리된 private object storage를 사용한다.

## 결과

원본 업로드 기능은 보안 정책과 함께 하나의 수직 기능으로 설계해야 한다.
